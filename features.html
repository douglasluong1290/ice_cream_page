Order Features

Features

Navbar

  • Hover and highlight: When user hovers over the text (hyperlink to another page), it will automatically change colour, and when user clicks on it, it permanently highlights the hyperlink of the page until switching to another page.
    Feature link: Navbar
  • Navbar toggle button: It only appears on mobile devices, initially the navbar pages are hidden, if the button is pressed on, the navbar pages will display in the column.
    Feature link: Toggle navbar button

Product

  • Hover: On PC, when a user hovers over one of the product containers, it will change background color.
    Feature link: Product hover
  • "Order Now" button: When a user clicks on the "Order Now" button, it will redirect to the Order page with the product option selected.
    Feature link: "Order Now" button

Order

  • Same as delivery address: When a user clicks on the checkbox with the address available in the input, it will automatically fill in the billing address input. If the address in the input is empty, it will show a warning.
    Feature link: Same as delivery address checkbox
  • Character limit depending on card distributors: When a user wants to use Visa and MasterCard for checkout, the card number will be limited to 16 digits, but when it comes to American Express, it will be 15 instead.
    Feature link: Card number
